# How to contribute

## Installation

Before you contribute make sure you install all necessary dependencies using:

```bash
$ composer install
```

## Run Tests

To run tests simply run the `phpunit` executable in the `vendor/bin`

```bash
$ ./vender/bin/phpunit
```

You should get an output similar to this

```bash
$ ./vendor/bin/phpunit
```
